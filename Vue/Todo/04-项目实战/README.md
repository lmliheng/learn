# Vue.js 项目实战

> 建议学习时长：15-30 天

---

## 概述

通过完整项目实战，将 Vue.js 知识综合运用，提升前端开发能力。

---

## 项目类型

### 1. 管理系统

- [ ] 技术栈：Vue 3 + Vue Router + Pinia + Element Plus
- [ ] 功能：用户管理、权限控制、CRUD

### 2. 电商平台

- [ ] 技术栈：Vue 3 + Vant
- [ ] 功能：商品列表、购物车、订单

### 3. 博客前台

- [ ] 技术栈：Vue 3 + SSR (Nuxt)
- [ ] 功能：文章展示、评论、标签
