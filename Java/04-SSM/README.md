## 重点概念

### IoC (Inversion of Control)
控制反转，将对象的创建权交给 Spring 容器来管理。

### DI (Dependency Injection)
依赖注入，Spring 容器动态注入对象所需的依赖资源。

### AOP (Aspect Oriented Programming)
面向切面编程，在不修改源码的情况下增强功能。
