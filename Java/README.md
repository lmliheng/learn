## 二、学习阶段与时长

| 阶段 | 内容 | 建议时长 | 重要程度 |
|:---|:---|:---:|:---:|
| **1. 基础** | 基本语法、面向对象、集合、IO流、多线程、异常 | 30 天 | ⭐⭐⭐⭐⭐ |
| **2. JavaWeb** | HTML/CSS/JS、Tomcat、Servlet、JSP、JDBC、Session | 10 天 | ⭐⭐⭐⭐ |
| **3. 数据库** | MySQL 安装配置、SQL语句、多表查询、索引、事务 | 5-7 天 | ⭐⭐⭐⭐ |
| **4. SSM** | Spring + SpringMVC + MyBatis | 7 天 | ⭐⭐⭐⭐ |
| **5. Spring Boot** | 快速开发框架、自动配置、Web开发 | 3-5 天 | ⭐⭐⭐⭐⭐ |
| **6. 微服务** | Spring Cloud、Redis、MQ、Docker、Elasticsearch | 10 天 | ⭐⭐⭐⭐ |
| **7. 项目实战** | 尚医通、尚筹网等完整项目练习 | 15-30 天 | ⭐⭐⭐⭐⭐ |

## 四、必学核心技术栈

```
基础核心:    Java基础 → 面向对象 → 集合 → IO流 → 多线程 → 异常
数据库:      MySQL → JDBC → 连接池(HikariCP)
前端基础:    HTML → CSS → JavaScript → jQuery
主流框架:    Spring → SpringMVC → MyBatis → SpringBoot
微服务:      SpringCloud → Redis → MQ消息队列 → Docker
版本控制:    Git → GitHub
```

---

## 五、学习建议

- ### 🎯 重点掌握
  Java基础语法、面向对象、集合、Spring Boot、MyBatis

- ### 💻 多敲代码
  > "光学不练假把式"，边学边写才能真正掌握

- ### 🚀 做项目
  学完框架后一定要做1-2个完整项目

- ### 📝 勤笔记
  好记性不如烂笔头

- ### 🤝 多交流
  遇到问题先自己解决，解决不了再问别人
