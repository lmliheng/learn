## 项目亮点总结

在面试中，项目亮点可以从以下方面总结：

1. **技术亮点**
   - 使用了哪些核心技术
   - 解决了什么技术难题

2. **业务亮点**
   - 复杂的业务逻辑
   - 高并发处理

3. **性能优化**
   - 数据库优化
   - 缓存使用
   - 异步处理

4. **项目架构**
   - 微服务拆分
   - 设计模式应用

## 面试常问问题

- 请介绍一下这个项目
- 项目的技术选型是什么？
- 遇到了哪些技术难点？如何解决的？
- 项目中使用了哪些设计模式？
- 如何保证数据一致性？
- 如何处理高并发？
