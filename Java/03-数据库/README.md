# 数据库 (MySQL)

> 建议学习时长：5-7 天

---

## 概述

数据库是 Java 开发中不可或缺的一部分，几乎所有的 Java 应用都需要与数据库进行交互。MySQL 是最流行的开源关系型数据库。
