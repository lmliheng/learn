表单

form标签的属性有：action提交表单数据的URL
  • action：提交表单数据的URL
  • method：提交表单数据的方法（GET/POST）
  • enctype：指定表单数据的编码类型（multipart/form-data）

登录表单:


input type="text"的属性有
  • name：为输入框指定名称
  • value：设置输入框的默认值,写入表单数据
  • required：指定输入框是否必填
  • placeholder：设置输入框的空白时的占位符
  • pattern：指定输入框的正则表达式
  • minlength：指定输入框的最小值
  • maxlength：指定输入框的最大值
  • accesskey：指定鼠标聚焦输入框的快捷键,如x，快捷键是alt+x
  • autofocus：指定输入框是否自动获得焦点
  • autocomplete：指定输入框是否自动完成,历史输入信息
input type="password"的属性有
input type="checkbox"的属性有
  • name：为复选框指定名称
  • value：设置复选框的默认值,写入表单数据
  • checked：指定复选框是否默认选中

上传文件表单:



input type="file"的属性有，通过css隐藏可以实现其它组件click事件触发文件上传
  • name：为文件上传框指定名称
  • accept：指定文件上传框接受的文件类型 accept=".jpg,.exe,.zip" 表示接受所有图片文件
  • multiple：指定文件上传框是否可以选择多个文件

含下拉框和文本域的表单:





input type="textarea" 用于大文本输入，比如评论，文章，反馈
  • name：为文本域指定名称
  • placeholder：设置文本域的空白时的占位符
  • rows：指定文本域的行数
  • cols：指定文本域的列数
label辅助标签的属性有，关联可以for-id 也可以用label标签包含目标函数
  • for：指定标签关联的元素的id属性值，用于关联输入框和标签，比如label click事件触发关联的输入框获得焦点，复选框勾选
  • access：指定标签关联的输入框的快捷键,如x，快捷键是alt+x
