> 建议学习时长：15-30 天

## 概述

通过完整项目实战，将 Node.js 知识综合运用，提升开发能力。

---

## 项目类型

### 1. 博客系统

- [ ] 技术栈：Express + MongoDB + EJS
- [ ] 功能：文章发布、分类、评论

### 2. 管理系统 API

- [ ] 技术栈：Express + MySQL + JWT
- [ ] 功能：用户管理、权限控制

### 3. 电商后端

- [ ] 技术栈：Node.js + Redis + MySQL
- [ ] 功能：商品管理、订单处理、支付接口
